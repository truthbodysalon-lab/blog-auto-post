# SEO / MEO ルールブック（整体院トゥルース ブログ自動投稿）
> 最終更新: 2026-04-29

## 基本方針
「長岡市 × 症状 × 体験意図」でロングテールキーワードを獲得。  
**6ステップ記事制作方式**で品質を担保しながら毎日10記事を自動生成・投稿する。

---

## ✅ 6ステップ記事制作方式（プロンプトに組み込み済み）

| ステップ | 内容 | 出力先 |
|---------|------|--------|
| Step 1 | タイトル候補5本 → SEO最優秀1本を選定 | title フィールド |
| Step 2 | 5ブロックアウトライン作成 | bodyHtml の構造 |
| Step 3 | ブロック1〜4の記事文章作成 | bodyHtml |
| Step 4 | SEO・読みやすさで修正 | bodyHtml（最終化） |
| Step 5 | ブロック5まで完成 | bodyHtml（全文） |
| Step 6 | まとめ文作成 | bodyHtml（締め・CTA） |

---

## 📋 自動生成される記事のSEO設定（全項目）

| 項目 | 仕様 | フォームフィールド |
|------|------|-----------------|
| **ナビ表示名** | 20字以内 | `Newpages[name]` |
| **スラッグ** | `nagaoka-{症状}-{切り口}-{YYYYMM}` 形式 | `Newpages[slug]` |
| **タイトル** | 35字以内・長岡市+主軸KW必須 | `Newpages[title]` |
| **h1テキスト** | 40字以内・タイトルと異なる表現 | `Newpages[h1_text]` |
| **ディスクリプション** | 120〜180字・院名+ベネフィット | `Newpages[description]` |
| **keywords** | 5〜8個カンマ区切り | `Newpages[keywords]` |
| **タグ** | 既存タグから3〜5個 | 管理画面タグ欄 |
| **カテゴリ** | 整体のみ（固定） | カテゴリチェックボックス |
| **公開設定** | 公開（即時 or 予約） | `Newpages[display_flag]` |

## 📝 本文構造ルール（5ブロック必須）

```
ブロック1：冒頭フック（読者の悩みへの共感・問いかけ）
ブロック2：原因解説（背骨の歪み・自律神経・季節の影響）
ブロック3：NG行動 or セルフケア方法（具体的・実践的）
ブロック4：整体院トゥルースの根本改善アプローチ（院の強みを自然に紹介）
ブロック5：まとめ + 来院CTA + NAP情報
```

**文字数：** 1800〜2500字（2000字前後が理想）  
**h2：** 3〜4個 / **h3：** 各ブロックに1〜2個

---

## 🏪 NAP情報（末尾CTAに毎記事必須）

```
■ 整体院トゥルース（ボディコーディネートサロン Truth）
■ 所在地: 新潟県長岡市
■ 専門: 肩こり・頭痛・姿勢・骨盤矯正の根本改善整体
■ 初回限定: 通常12,000円 → 4,680円（税込・施術40分・1日3名限定）
```

## 🚫 禁止語・推奨語

**禁止：** 治る / 治療 / 絶対 / 100% / 最高 / No.1 / 医学的に証明 / ビフォーアフター画像  
**推奨：** 改善 / 整える / サポート / 根本から / 繰り返さない体に / 寄り添う

---

## 🔁 重複防止ルール

- 過去20投稿のタイトルを記録（`posts/history.json`）
- 80%以上の類似タイトルは自動検出して再生成（最大2回）
- スラッグは `nagaoka-{症状英語}-{切り口英語}-{YYYYMM}` で自動生成
